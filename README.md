# Node Crash Course Files

> These files go with the Node crash course on YouTube.

## Quick Start

- Download the latest version of node :
- https://nodejs.org/en/download

```bash
# Install dependencies
npm install

# Serve on localhost:5000
npm start
```
node index.js 
- http://localhost:5000/
